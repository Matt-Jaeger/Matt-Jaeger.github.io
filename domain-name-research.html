Domain Name Research

  • What is an internet domain name?
  • An internet domain name is an identification string that gives the owner power over the administrative functions of the domain. The domain name creates a name for an IP address.
  • Why are domain names so important on the web?
  • Domain names are important because they allow us to give names to IP addresses. Instead of having to remember a bunch of digits for the websites we want to visit, we can just type in the name of the website.
  • Why do domain names have to be unique?
  • Domain names have to be unique because otherwise it would be chaos and confusion. You would not want others making websites as copycats of other big websites such as google.
  • What does it mean to "register" a domain name?
  • To register a domain name, you must contact a TLD (top level domain) to purchase a domain name for your website.
  • If you wanted a domain name that is already taken, how could you get it?
  • If you wanted to get a domain name that somebody already owns, you will have to buy it from them. If you cannot or do not want to buy it from them you can pick a different name or pick a different domain extension (.net, .tech, .org).

Project plan

Title

Phoenix Board Game Cafe>

Goal

The purpose of the site is to attract people to a brick a mortar board game store and restaurant. The site will also show information about events that take place in the store, any deals that are going on or coming up, and have resources for people who are looking to find a group to play games with.

Target Audience

The site will be for anyone who loves board or card games or has an interest in trying them out for the first time. The main audience will like be families, the surrounding community, teenagers, and people in the 20-40 age range, but nothing about the store or the features on the site will exclude people outside those ranges.

Main Features

The website will include a gallery of pictures of the store, the menu and taplist, a shop page for buying games or cards, a list of upcoming store events, a calender that will list events much further out, and a contact form/contact information for the shop.

Personality

Warm, welcoming, family/community oriented, appealing, concise.

The site should welcome people to come in and check out the store, while easy navigation should let people find what they are looking for quickly.

Limiting Factors

  • Expect people to have decent internet connections, there will be quite a few pictures.
  • Expect people to have some comfort with techonology
  • Would expect users to visit on mobile devices
  • No idea how to do a secure shop page, probably something you could make work through a web hosting service as I am sure thats a very requested feature.
  • Would need to learn more about databases to manage the board game and card game inventories online

Styles guide

Background

Background is brown

{background-color: #803129}

Default font

Text will be a combination of white, black, and orange, 14 pixels

p { font-size: 14px; color: #FF6452;}

Header

The header will be 300px with a centered logo. The background will be the reddish-orange color.

header {
height: 300px;
background-color: #CC4F41;
background-image: url("/images/logo.png");
background-repeat: no-repeat;
background-position: center;
background-size: cover;
}

Navigation

The main navigation is 150px tall, positioned just below the header with a background color of faded brown with orange text.

nav { height: 120px; background-color: #80544F; color: #FF6452; }

Images

Images will be on the pink background with a 2px border.

figure { background-color: #FFA89E; border: 2px solid #CC4F41; }

img { padding: 1em; }

Links

Links will be the reddish orange with the orange being used when hovered.

a { color: #CC4F41 }

a:hover { background-color: #6699CC; border 2px dotted #FF6452; }
